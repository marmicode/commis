# Install pnpm

https://pnpm.io/installation

```sh
corepack enable
```

or if you are using [Volta](https://volta.sh/)

```sh
volta install pnpm
```

# Retrieve source code and install dependencies

```sh
git clone git@github.com:marmicode/whiskmate.git --branch testing-000-boilerplate

cd whiskmate

pnpm install
```
